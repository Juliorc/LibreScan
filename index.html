Acerca de Librescan

LibreScan es un proyecto de Software Libre desarrollado en el Laboratorio Experimental del Centro Académico de Alajuela, sede del Instituto Tecnológico de Costa Rica. Este proyecto de investigación consiste en la elaboración de un software que agrupa distintas herramientas que cumplen funciones específicas dentro del flujo de tareas necesarias para escanear un documento. El objetivo de LibreScan es promover el acceso libre al conocimento y la educación, al proveer un software libre y gratuito que permita realizar el proceso de captura de fotos, procesamiento y generación de un documento digital.

Este proyecto está ligado al movimiento DIY Book Scanner, quienes se encargan de la elaboración de escáneres artesanales. LibreScan complementa esta iniciativa al aportar un software que automatiza el proceso de escaneo con este tipo de dispositivos.

Proceso

A continuación se listan los pasos que conforman el flujo de escaneo:

  • Capturar las fotos haciendo uso de cámaras Canon y CHDKPTP.
  • Edición de las imágenes con ScanTailor para la generación de archivos tif.
  • Reconocimiento de texto con Tesseract OCR, produciendo archivos hocr.
  • Ensamblaje del producto final. Actualmente está soportado únicamente PDF, generado utilizando la herramienta PDFBeads.

Instalación y uso

Para instalar y hacer uso de LibreScan se deben seguir los pasos listados en el README del repositorio de GitHub
